TASK CENTRAL

Welcome to Task Central – a simple, modern, and mobile-friendly way to keep your tasks under control.


Task Central is a web app for managing your tasks.

Features

- Add, edit, and delete tasks
- Organize by category and priority
- Search and filter – quickly find what you need.
- Mark tasks as complete
- Responsive design – works great on your phone, tablet, or desktop.
- Dark/light mode
- Handy notifications



How to use it

- Add a new task using the form on the left.
- Edit or delete any task with the buttons on each task card.
- Filter or search using the controls above your task list.
- Mark tasks as complete by clicking the circle button.
- Switch themes with the toggle in the header.
- Your tasks are saved in your browser, so you can close the tab and come back later.

- Built with vanilla JavaScript (no frameworks required)
- Uses localStorage to save your tasks
- Mobile-first CSS for a smooth experience on any device
- No accounts, no tracking, just your tasks
